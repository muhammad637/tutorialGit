Git Branch and Merge

  1. membuat branch baru

    git branch (nama-branch)

  2. Melihat semua branch yang ada

    git branch

  3. Melihat semua perubahan yang ada di git

    git log --all --decorate --online --graph

  4. create alias (untuk menyimpan perintah ke nama perintah yang baru)

    5. alias init="git init"

  5. masuk ke branch

    git checkout (nama-branch)

  6. Menggabungkan branch

    git merge (nama-branch yang mau digabungkan)

  7. Git Rebase(memajukan branch disini(branch saat ini) dari disana(brach awal)) tapi wajib rebase balik dari branch awal ke branch saai ini

    git rebase (nama-branch)
